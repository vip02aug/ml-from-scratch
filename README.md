# ML From Scratch

This repository contains implementations of Machine Learning algorithms from scratch.

## Goals
- Understand ML mathematically
- Implement algorithms without libraries
- Build strong foundations for research

## Topics Covered
- Linear Regression (in progress)
- Logistic Regression (upcoming)
- Naive Bayes (upcoming)

## Structure
Each topic contains:
- Implementation (Jupyter Notebook)
- Explanation (README)
- Results (plots)
